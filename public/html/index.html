Available Functionality of this Node API

Get: /Users
Description: Should return a list of all users and their properties as a JSON array. Passwords should be hidden.
Success: Code 200 OK Success status
Example: [ { "userId":1, "firstName":"Jon", "lastName":"Snow", "emailAddress":"jsnow@gmail.com", "password":"hidden" }, { "userId":2, "firstName":"Killua", "lastName":"Zoldyck", "emailAddress":"kzoldyk@gmail.com", "password":"hidden" }, { "userId":3, "firstName":"Paul", "lastName":"McCartney", "emailAddress":"pmccartney@gmail.com", "password":"hidden" } ]


Post: /Users
Description: Should Create a new User record and take in the following properties: First Name, Last Name, and Email & returns the new user.
Success: Code 201 Created Success status
Example: { "userId": 4, "firstName": "Maya", "lastName": "Angelou", "emailAddress": "mangelou@gmail.com", "password":"1234567" }
Failure: Code 409 Conflict response status Example: {message:"Duplicate userID"}


Get: /Users/{userId}
Description: Should return a specific User by the user ID provided in the URL with all its properties if it exists.
Success: Code 200 OK Success status
Example: { "userId": 3, "firstName": "Paul", "lastName": "McCartney", "emailAddress": "pmccartney@gmail.com", "password":"123abc" }
Failure: Code 404 User not Found
Example: {"message": "User with id 8 was not found."}


Patch: /Users/{userId}
Description: Should allow you to Update any data for an existing User by the user ID provided in the URL, except the ID field & returns the updated user.
Success: Code 200 OK Success status
Example: { "userId": 4, "firstName": "Maya2", "lastName": "Angelou2", "emailAddress": "mangelou@gmail.com2", "password":"1234567" }
Failure: Code 404 User not Found
Example: {"message": "User with id 5 was not found."}


Delete: /Users/{userId}
Description: Should delete a user if it exists by the user ID provided in the URL.
Success: Code 201 Created Success status
Example: {"message": "User with id 4 has been deleted."}
Failure: Code 404 User not Found
Example: {"message": "User with id 5 was not found."}
